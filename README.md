**Expense Tracker** is an Expense Tracker App made for Android devices.
## Features
- Create new payment entry
- Stores all payments to be done using firebase
- Tap on one of the payment entry to view details 
- You can pay using GPay and Paytm
- Stores all recently done transactions using firebase

## Installation

1. **Clone the Project:**
    1. Run ` git clone https://github.com/ujjman/ExpenseTracker ` in terminal
    1. Android Studio -> File -> Open
    
## Firebase

Expense Tracker uses [Firebase](https://firebase.google.com/) to store all payments and user information.
## Libraries

- [Firebase](https://firebase.google.com/) - For storing data
- [Cuberto](https://github.com/Cuberto/liquid-swipe-android) - For animations between fragments
- [Lottie animations](https://github.com/airbnb/lottie-android) - To show animated data
